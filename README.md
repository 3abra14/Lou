# Lou's Birthday Card 💜

A premium, interactive birthday card web application designed with "Antigravity" aesthetics.

## Features
- ✨ **Glassmorphism Design**: Sleek, translucent UI elements with deep blurs.
- 💜 **Interactive Animations**: GSAP-powered transitions and mouse-responsive tilt effects.
- 💖 **Dynamic Effects**: Anime.js heart rain and mouse glow.
- 💌 **Personal Message**: A beautiful dedicated section for the birthday message.

## How to View
Simply open `index.html` in any modern web browser.

## Built With
- HTML5 / CSS3 (Vanilla)
- [GSAP](https://greensock.com/gsap/) (Animations)
- [Anime.js](https://animejs.com/) (Particles)
- [Google Fonts](https://fonts.google.com/) (Outfit & Pacifico)
